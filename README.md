# 🌐 Proyecto Final: Despliegue Automatizado y Monitoreo de un CMS (WordPress)

## 👥 Integrantes
* **Cristian Steven Castro Ortega** - Ingeniería de Sistemas

## 🎯 Objetivo del Proyecto
Este proyecto tiene como finalidad diseñar e implementar un entorno web multicapa utilizando **WordPress** como servicio principal. Todo el ecosistema estará montado sobre contenedores de **Docker**, automatizado mediante scripts de **Bash** y supervisado a través de **Prometheus y Grafana**.

## 🛠️ Stack Tecnológico Seleccionado
* **Virtualización:** Docker & Docker Compose
* **Automatización:** Bash Scripts (`.sh`)
* **Monitoreo:** Prometheus + Grafana
